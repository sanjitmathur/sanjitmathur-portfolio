INTERACTION SYSTEMS (VERY IMPORTANT)

Use these interaction systems intentionally.

1. PARALLAX DEPTH SYSTEM

Create multiple depth layers.

Foreground objects move faster than background objects.

Mouse movement slightly rotates the camera.

The environment must feel dimensional.

---

2. PHYSICS-BASED HOVER EFFECTS

Interactive elements should react with realistic motion.

Examples:

• project cards tilt in 3D
• lighting changes based on cursor position
• depth shadow changes dynamically

Use spring-based easing instead of linear animation.

---

3. CURSOR-DRIVEN INTERACTION

The cursor should influence the environment.

Examples:

• light source follows cursor slightly
• nearby elements shift subtly
• particles react to cursor proximity
• UI panels tilt slightly toward cursor

Cursor interactions must feel subtle and elegant.

---

4. ENTRANCE REVEALS

Elements should emerge from depth rather than appear suddenly.

Examples:

• text slides forward in Z-space
• panels reveal through masked motion
• staggered element reveals

Motion should feel cinematic.

---

5. MICRO INTERACTIONS

Every clickable element must give feedback.

Examples:

• buttons compress slightly when clicked
• links animate underline
• icons glow softly on hover
• panels shift depth slightly

These details create a premium feel.
